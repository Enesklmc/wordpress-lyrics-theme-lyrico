Lyrico Documentation

The Lyrico theme for WordPress made for everybody that wants to have a Lyrics website, featuring Lyrics, Artists, Albums, Playlists and connections between each other. Lyrico has the features that should be on a lyrics website, including responsive design and a lot of customization options.

Lyrico requires the latest WordPress version (5.3 or later) and PHP (7.1 or later).

Let's get started

Install Lyrico

1 Login to your WordPress dashboard(wp-admin).

2 Click Appearance > Themes from the menus on the left.

3 Now you will see your installed Themes. Click on Add New button at the top of the themes.

4 Then click on Upload Theme button (in the same place as the Add New button) and a Choose File will appear.

5 Click on the Choose File button and browse to the lyrico.zip file inside the downloaded and extracted package(lyrico-files.zip).

6 Then click on Install Now and wait for the installation to finish. Then click on Activate to activate Lyrico.

Install Lyrico Plugin

- Lyrico Plugin comes within the lyrico.zip file.The only thing you should do is activate the Lyrico Plugin.

1 After you installed Lyrico Theme required plugins will appear top of your screen.

2 Click Begin activating plugin in the notification window.

3 After Install Required Plugins page opened, click Activate under the Lyrico Plugin. Done.

4 Now, go Dashboard and when you check the Admin menus you will see Lyrics, Albums, Artists, Playlists, If the plugin was installed successfully.

- Lyrico Plugin will activate:

  • Post types: Lyrics, Albums, Artists, Playlists
  • Custom meta boxes
  • Custom Taxonomy: Genres
  • Lyrico Latest Posts Widget
  • Gutenberg Playlist Item block

Install Starter Content

Starter content will make your website look like http://lyrico.klmcthemes.com/.

It will add Pages, Lyrics, Artists, Albums, Playlists, Images and Menu. You can edit them later.

1 Go Tools -> Import from Admin menus.

2 Find WordPress from the list.

3 Click on Install Now, then Run Importer.

4 Choose starter_content.xml file from Lyrico Package(lyrico-files) and click Upload file and Import button.

5 Don't forget to activate Download and import file attachments option for Images.

Creating Template Pages

1 Create a new page from Pages -> Add New

2 Add Title, example : Homepage.

3 Select Lyrico Homepage Template from the Page Attributes Section right of the editor.

Templates
Genres:

Shows All created genres.

Lyrico Home page template.

Albums:

Shows Popular and All Albums. (customizable).

Artists:

Shows Popular and All Artists. (customizable).

Playlists:

Shows Popular and All Playlists. (customizable).

Sidebarless Default Page:

It creates an empty page without sidebar.

Setup Front Page

1 Go to Settings -> Reading.

2 Select A static page.

3 Then select the page you've created with homepage template "Home" for Homepage.

4 And select the page you've created with Default page template "Blog" for Posts Page.

Activate Menu

1 Go to Appearance -> Menus.

2 Write your Menu Name, then click Create Menu.

3 Select pages you've just created and Add to Menu.

4 Select Display location from Menu Settings. Show your menu in Top Menu or Footer.

Setup Homepage

After Setup Front Page and activate Lyrico Homepage Template as your front page, you can customize homepage from Customizer.

You should select Playlists Page from Select Playlists Page setting (also Select Artists Page as your artists page, Select Albums Page as your albums page, Select Blog Posts Page as your latest post page).

The page to which the button sends the user needs to be selected, otherwise buttons are not gonna work.

Customize Lyrico

All customization settings are found in the Appearance -> Customize section on Admin Menu.

Settings based on the currently previewed page

These settings appear according to the currently previewed template.

For Example:

When you look at Artists Page in your customizer screen there will be Artist Layout & Settings section in your customizer.

These pages have additional settings that will be appear when you change the page from customizer: Home Page, Playlists, Artists, Albums, Single Lyrics, Single Playlist, Single Artist, Single Album, Genres.

Creating Lyrics

Don't forget to create Artist and Album before creating lyrics. You'll need to select them when you create lyrics.

- Create Artist
  • From your WordPress Admin menu go Artists -> Add New.
  • Add title to be the name of the Artist.
  • You can fill out some forms that give information about the artist in About Artist section.
  • Select related genres of artist from the Genres Section in the editor sidebar.
  • Select Tags wisely for show related Artists in single artist page. It allows users to discover similar artists.
  • Choose Thumbnail Image of the Artist.
  • Publish it.

*You can add extra content to the top of the Artist page using the Gutenberg/Classic Content Area.

- Create Album
  • From your WordPress Admin menu go Albums -> Add New
  • Add title to be the name of the Album.
  • Select Primary Artist from the About Album section.
  • Select related genres of album from the Genres Section in the editor sidebar.
  • Select Tags wisely for show related Albums in single album page. It allows users to discover similar albums.
  • Choose Thumbnail Image of the Album.
  • Publish it.

*You can add extra content to the top of the Album page using the Gutenberg/Classic Content Area.

- Create Lyrics
  • From your WordPress Admin menu go Lyrics -> Add New
  • Add title to be the name of the Song.
  • Add Lyrics in Editor area.
  • Select Primary Artist, Other Artists(if exists) and Album from the About Lyrics section.
  • Select related genres of the Lyrics from the Genres Section in the editor sidebar.
  • Select Tags wisely for show related Lyrics in single lyrics page(Genres also affects it). It allows users to discover similar Lyrics/Songs.
  • You can fill out some forms(Details, Links) that give information about the lyrics/song in About Lyrics section.
  • Publish it.

Creating Playlist

  • From your WordPress Admin menu go Playlists -> Add New
  • Add title to be the name of the Playlist.
  • Add Playlist items in Editor area.
  • Select related genres of the Playlist from the Genres Section in the editor sidebar.
  • Select Tags wisely for show related Playlists in single Playlist page(Genres also affects it). It allows users to discover similar Playlists.
  • Publish it.
Adding Playlist Items with Custom Gutenberg Block

Playlist Item block supports schema.org

1 In Gutenber Editor click on the Add Block button and select Common Blocks > Playlist Item.

2 Click on Playlist Item button from right-top of the block to trigger the block.

3 Block fields will appear right side of the editor.

4 You can also add button, description or embed inside the playlist item block by click on Plus button inside the block.

Widget: Lyrico Posts

Add Lyrico Posts Widget to your sidebar from your WordPress Admin menu go Appearance -> Widgets.

In Available Widgets section the *Lyrico Posts Widget will be on the top. Drag and drop widgets to the place you want to add.
